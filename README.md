# 网页版贪吃蛇游戏

一个使用 HTML5 Canvas + JavaScript 实现的贪吃蛇游戏。

## 功能特点

- 经典贪吃蛇玩法
- 支持键盘控制（方向键或 WASD）
- 长按方向键加速功能
- 计分系统
- 游戏设置（声音、难度）
- 响应式设计

## 如何运行

1. 克隆仓库：
```bash
git clone [仓库URL]
```

2. 直接在浏览器中打开 `index.html` 文件即可开始游戏

## 游戏控制

- 方向键或 WASD：控制蛇的移动方向
- 长按方向键：加速移动
- 点击"设置"：调整游戏选项

## 技术栈

- HTML5
- CSS3
- JavaScript (原生)
